🛠️ Recuperación de Emergencia

Panel de recuperación para el Sistema de Inventario ADDBOX

📡 Diagnóstico de Conexión Pendiente

Presiona "Diagnosticar" para verificar el estado de Supabase...

👤 Crear Usuario Admin de Emergencia

Crea un usuario directamente en Supabase Auth para recuperar el acceso al sistema.

Completa los datos y presiona "Crear Usuario Admin"...

📋 Verificar Tablas

Lista las tablas disponibles en Supabase y verifica que los datos existen.

Presiona "Verificar Tablas" para listar las tablas disponibles...

⚙️ Corregir Estado de Instalación

Si el sistema te redirige constantemente a la página de instalación, ejecuta esto.

Presiona "Marcar como Instalado" para corregir el estado...

📖 Instrucciones Rápidas

  1. Paso 1: Ve a supabase.com/dashboard y reactiva el proyecto (si está pausado).
  2. Paso 2: Usa el botón "Diagnosticar Conexión" para verificar si Supabase responde.
  3. Paso 3: Si la conexión funciona, crea un usuario admin con "Crear Usuario Admin".
  4. Paso 4: Usa "Marcar como Instalado" para evitar redirecciones.
  5. Paso 5: Ve a Iniciar Sesión y prueba las credenciales.
