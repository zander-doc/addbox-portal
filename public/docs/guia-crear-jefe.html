📖 Guía: Crear Cuenta del Jefe

Instrucciones paso a paso para configurar la primera cuenta administrativa de ADDBOX LLC

⏱️ Tiempo estimado: 5 minutos
1

Generar la Master Key

El desarrollador debe generar una clave maestra que será entregada al jefe/dueño para crear su cuenta.

/admin-dashboard/setup/dev-master-key.html
localhost/admin-dashboard/setup/dev-master-key.html

🔑 Generar Master Key (Desarrollador)

Esta clave se usará una sola vez para que el jefe cree su cuenta.

⚠️ Importante: Esta clave no se puede recuperar después de cerrar esta página.
2

Guardar la Master Key

Después de hacer clic en "Generar", aparecerá la clave. Debes guardarla inmediatamente porque no se puede recuperar.

localhost/admin-dashboard/setup/dev-master-key.html

✅ Master Key Generada

ADDBOX-OWNER-a3f2b1c4-d5e6f7a8-b9c0d1e2-f3a4b5c6

Después de generar la clave, puedes ir a crear la cuenta del jefe.

Opciones para guardar:

  • 📥 Hacer clic en "Descargar" → se guarda como addbox-owner.key
  • 📋 Seleccionar la clave y copiar (Ctrl+C) → pegar en lugar seguro
  • 📸 Tomar captura de pantalla como respaldo
3

Entregar la Master Key al Jefe

El desarrollador entrega la clave al jefe/dueño de la empresa. Métodos recomendados:

  • 🤝 En persona (más seguro)
  • 🔒 Por mensaje cifrado (Signal, WhatsApp con desaparición)
  • 📧 Por email (menos seguro, pero funcional)
  • 📁 Compartir el archivo .key por medio seguro
4

Crear la Cuenta del Jefe

El jefe/dueño accede a la página de creación de cuenta e ingresa sus datos junto con la master key.

/admin-dashboard/crear-jefe.html

Confirmación y Redirección

Si todo es correcto, verás un mensaje de éxito y serás redirigido al login en 2 segundos.

👤 Crear cuenta del Jefe

✅ Cuenta del jefe creada correctamente. Redirigiendo al login...

Lo que sucede internamente:

  1. ✓ Se verifica la master key contra el hash en la base de datos
  2. ✓ Se crea la cuenta en Supabase Auth (email + contraseña)
  3. ✓ Se inserta el perfil en la tabla usuarios con rol "jefe"
  4. ✓ Se marca first_run = false (setup completado)
  5. ✓ Se redirige al login
6

Solución de Problemas

Si encuentras algún error durante el proceso, consulta esta tabla:

ErrorCausaSolución
"Completa todos los campos" Un campo está vacío Verifica que todos los campos tengan contenido
"Contraseña muy corta" Menos de 6 caracteres Usa una contraseña de 6+ caracteres
"Master key incorrecta" La clave no coincide Verifica que copiaste la clave completa sin espacios extra
"Error al verificar instalación" Supabase no responde Verifica conexión a internet y que Supabase esté activo
"No se ha generado master key" Paso 1 no completado Ve primero a /setup/dev-master-key.html
"User already registered" Email ya existe en Auth Usa otro correo o elimina el usuario en Supabase Dashboard
"Error al guardar perfil" Tabla usuarios con restricciones Verifica que la tabla existe y las políticas RLS permiten insert
🔒

Notas de Seguridad

  • La master key es de un solo uso. Una vez creada la cuenta, ya no sirve.
  • El hash SHA-256 es irreversible. Nadie puede obtener la clave desde la base de datos.
  • Después del setup, first_run = false impide repetir el proceso.
  • El rol "jefe" tiene acceso total al sistema.
  • No compartas la master key por canales inseguros.
