2. Rutas absolutas

Estándar oficial de navegación

ADDBOX PRO usa rutas absolutas para evitar errores de navegación:

  • /admin-dashboard/modules/dashboard/dashboard.html
  • /admin-dashboard/modules/usuarios/usuarios.html
  • /admin-dashboard/modules/inventario/productos.html
  • /admin-dashboard/inicio-de-sesion.html

Regla general:

/admin-dashboard/... (siempre absoluto)

3. Módulos del sistema

Organización por áreas funcionales

Los módulos están organizados por carpetas dentro de /admin-dashboard/modules/:

  • dashboard/ — panel principal del jefe
  • usuarios/ — gestión de usuarios internos
  • inventario/ — productos, stock, movimientos
  • movimientos/ — entradas y salidas
  • reportes/ — informes y estadísticas

4. Componentes reutilizables

Elementos compartidos en todo el sistema

Los componentes se encuentran en /admin-dashboard/components/:

  • sidebar.html — menú lateral
  • header.html — encabezado superior

Estos componentes se cargan dinámicamente mediante JavaScript.

5. Servicios internos

Funciones JS que manejan lógica del sistema

Los servicios están en /admin-dashboard/assets/js/:

  • auth.js — login, logout, validación de sesión
  • supabase.js — conexión y consultas
  • utils.js — funciones auxiliares
  • router.js — carga de módulos y navegación

6. Estándares de CSS

Reglas para mantener consistencia visual
  • Siempre usar /admin-dashboard/assets/css/ como ruta base.
  • Evitar rutas relativas como ../../css salvo en casos controlados.
  • Usar clases con prefijo app- para elementos globales.
  • Evitar estilos inline salvo en mockups o documentación.

7. Estándares de JavaScript

Buenas prácticas internas
  • Usar async/await para todas las operaciones con Supabase.
  • No mezclar lógica de UI con lógica de datos.
  • Centralizar errores en utils.js.
  • Usar nombres descriptivos para funciones y variables.

8. Navegación interna

Cómo se mueve el usuario dentro del sistema

La navegación se basa en:

  • Sidebar (component)
  • Rutas absolutas
  • Validación de sesión
  • Validación de rol
